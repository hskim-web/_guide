    SCSS 설정

    Extend 설치

    View > Extensions (Ctrl + Shift + X)

    1. Live Sass Compiler 설치 (3, 4, 5 포함된 settings.json 전체 소스 아래 첨부)

    2. Sass Lint 설치

      • 설치 후 VSCode 상태 표시줄 Watch Sass 확인

    3. ' Live Sass Compile Config ' 검색

      File > Preferences > Settings > Extensions > Live Sass Compile Config

      1. Generate Map 설정

        Live Sass Compile › Settings: Generate Map

        "liveSassCompile.settings.generateMap": false

      2. Format 설정

        Live Sass Compile › Settings: Formats

        "liveSassCompile.settings.formats": [
    {
        "format": "compressed",
        "extensionName": ".min.css",
        "savePath": "~/../css"
    },
    {
        "format": "expanded",
        "extensionName": ".css",
        "savePath": "~/../css"
    }
]
        • format 종류
          • nested : 기본값, 포맷 미지정
          • expanded : 계층 구조로 정리
          • compact : 줄바꿈 없이 한 줄로 적용
          • compressed : 불필요한 공백 모두 제거

    Project 구조

    • /lib : 서버(Back End)와 통신하는 파일
    • /public/ : 클라이언트 측(Front End) 컴파일에 필요한 파일
      • /public/js : 클라이언트 측 js 파일
      • /public/css : SCSS로부터 자동 컴파일된 파일
      • /public/scss : SCSS 파일
      • /public/views : 메뉴 단위 구성 (서비스의 조합)
    • /routes : 이동 라우터
    • /service : 서비스 단위 구성
    • /template : 레이아웃
